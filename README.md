##Exploratory Data Analysis (EDA) - Recruitment Dataset 

**Project Overview**
This project focuses on performing a comprehensive exploratory analysis on recruitment data to identify patterns in hiring, candidate qualifications, and recruitment bottlenecks. The goal is to transform raw data into actionable insights that can improve hiring efficiency.

**Key Objectives**
* Data Cleaning: Handling missing values and ensuring data consistency for reliable analysis.
* Statistical Analysis: Computing key metrics like mean time-to-hire and candidate distribution.
* Visual Insights: Using charts to visualize demographics, skill distributions, and application trends.

**Tech Stack**
* Language: Python 3.x
* Libraries: Pandas (Data Manipulation), NumPy (Numerical Analysis), Matplotlib & Seaborn (Visualization)
* Environment: Jupyter Notebook
